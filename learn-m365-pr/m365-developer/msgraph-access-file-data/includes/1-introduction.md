OneDrive is the files hub for Office 365. OneDrive enables users to access and collaborate on files no matter where they're stored. Microsoft Graph enables developers to use a single API to work with the files in OneDrive. Files in Office 365 are stored in drives. Users can store files in a personal drive, their OneDrive, or in a shared drive powered by a SharePoint document library. In this module, youâ€™ll learn how to access files in OneDrive, both reading and writing files, and how to traverse relationships between files and users.

## Prerequisites

- Basic knowledge of REST services & APIs
- Basic knowledge of Microsoft Graph
- Ability to develop with ASP.NET Core at the intermediate level
- Experience using Visual Studio Code at the beginner level
- Access to a [Microsoft 365 tenant](https://developer.microsoft.com/office/dev-program?ocid=MSlearn)

## Learning objectives

By the end of this module, you'll be able to:

- Demonstrate how to retrieve a list of files OneDrive
- Demonstrate to download a file from OneDrive using a unique ID
- Get a list of files trending around the signed in user
- Demonstrate how to upload a large file to OneDrive
