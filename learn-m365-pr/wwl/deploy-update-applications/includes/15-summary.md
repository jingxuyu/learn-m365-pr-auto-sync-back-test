This module covered various ways to deploy and update applications across an organization. With app management in Intune, administrators should first understand the app lifecycle. Once application needs are scoped, Administrators have various deployment methods to choose from. Solutions such as Microsoft Intune or Microsoft Store for Business can be used to deliver applications to devices. If the device is domain joined, Configuration Manager and Group Policy can be used as well.
