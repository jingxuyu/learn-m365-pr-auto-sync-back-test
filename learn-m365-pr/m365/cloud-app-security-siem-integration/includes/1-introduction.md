You're an administrator for Contoso, a large multi-national organization. Contoso IT management are implementing a shift to Microsoft 365. As part of this process, it's important that IT can maintain control of critical corporate data. As a security administrator at Contoso, you're involved with planning for this data control. You've decided to implement Microsoft Cloud App Security and you are investigating implementing Azure Sentinel or another security information an event management (SIEM) solution.

## Learning objectives

After completing this module, you'll be able to:

- Connect Azure Sentinel to Microsoft Cloud App Security.
- Enrich incidents in Azure Sentinel with playbooks.
- Triage infrequent country requests with Microsoft Cloud App Security and Azure Sentinel.
- Connect a non-Microsoft security information and event management solution to Microsoft Cloud App Security.
