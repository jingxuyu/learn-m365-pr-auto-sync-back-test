As you have seen, using the Microsoft Threat Protection suite lets you identify, respond, and prevent most common security threats. You can now use Microsoft Secure Score to get an immediate and up-to-date view of your organization's security posture and take remedial action to reduce the threat and improve your overall score. Finally, you can utilize the Microsoft Intelligent Security Graph to leverage security analytics and machine learning to help your organization combat cybercrime.

In this module, you have learned how to:

* Identify common security threats.
* Discover how enterprises can prevent, detect, and respond to threats.
* Discover your security posture with Microsoft Security Center and Secure Score.
* Discover the value that Microsoft's Intelligent Security Graph and Azure Sentinel provides to secure organizations.
