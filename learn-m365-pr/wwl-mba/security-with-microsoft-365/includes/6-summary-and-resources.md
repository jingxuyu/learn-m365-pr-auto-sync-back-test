
As security threats grow and evolve, so does the cybersecurity landscape, with more devices, data, and networks to protect than ever. This rapid pace of change and acceleration can push IT teams to the limit. Microsoft 365 offers intelligent, built-in security that reduces complexity for IT, while keeping employees productive and connected from anywhere. 

Now that you have reviewed this module, you should be able to:

- Describe how Microsoft 365 security offerings work together to keep your data safe.
- Articulate the benefits of integrated security in Microsoft 365.
